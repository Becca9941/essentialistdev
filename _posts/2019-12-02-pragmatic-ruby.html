Regular expressions

A regular expression is simply a way of specifying a pattern of characters to be matched in a string.

  • /Perl|Python/ = match a string that contains either the word 'Perl' or the word 'Python'.
  • /ab+c/ = match a string that contains an 'a' followed by 1 or more 'b's followed by a 'c'.
  • line.sub(/Perl/, 'Ruby') = replace first 'Perl' with 'Ruby'.
  • line.gsub(/Python/, 'Ruby') = replace every 'Python' with 'Ruby'.

def call_block
  puts "Start of method"
  yield
  yield
  puts "End of method"
end
